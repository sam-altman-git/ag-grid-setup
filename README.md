# Project Name: Radin NextJs (formerly seppStreamNextjs)

## Overview

This project is a Next.js application for SEPP Stream.

## Scripts

The following scripts are defined in `package.json`:

- `dev`: Starts the development server in watch mode.
- `build`: Compiles the application for production deployment.
- `start`: Launches the production server.
- `lint:check`: Checks the codebase for linting issues using ESLint.
- `lint:fix`: Fixes fixable linting issues in the codebase using ESLint.
- `lint:inspect`: Inspects the codebase for detailed linting diagnostics using
  ESLint.
- `format:check`: Verifies code formatting using Prettier without making
  changes.
- `format`: Formats the codebase using Prettier.

## Installation

1. Clone the repository.
2. Run `yarn install` to install dependencies.

## Usage

- Run `yarn dev` to start the development server.

- Run `yarn build` and `yarn start` to start the production server.
